# ZELLIJ Store

Site statique pret pour GitHub Pages.

## Ajouter un nouvel article

1. Ajoute l'image du produit dans `assets/`.
2. Duplique l'objet dans `products.js`.
3. Garde le meme objet dans `products.json` si tu veux utiliser les donnees ailleurs plus tard.

## Publier sur GitHub Pages

1. Cree un repository GitHub.
2. Envoie le dossier `zellij-site`.
3. Dans GitHub: `Settings` -> `Pages` -> `Deploy from a branch`.
4. Choisis la branche `main` et le dossier racine.

Le site fonctionne sans framework ni installation.
